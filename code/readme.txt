## Steps to follow to run the quickstart app 

1. clone the app by running the command below : 
  $ git clone https://github.com/HyounesH/battlebrain2020.git
  
2. go the project :
  $ cd battlebrain2020\code\seats_reservation 
  
3. install required dependencies and start the project (backend & frontend)
  -- Backend
  
  $ mvn clean install
  $ mvn spring-boot:run
  
  -- Frontend
  
  $ cd src\main\frontend
  $ npm install
  $ ng serve
